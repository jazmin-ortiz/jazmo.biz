Jazmin's Bootstrap Website

Hi! I'm Jazmin. I'm a software engineer and this is my dumb little scrolly website.

If you want to learn more about me go to jazminortiz.com

Why did you make this?

Bootstrap websites are everywhere and they all have this same look.

The content may be different but the user experience gives it away.

It feels like the elements that bootstrap provides sort of pushes you to put all of the information into a single page.

I feel like every bootstrap website you end up swiping all around on a single page. So Scrolly!

It seems like the general consensus is that bootstrap makes it very easy to make a fairly polished website.

I want to see how easy bootstrap is and make a dumb little scrolly website like all the ones I see online.
